About Our Team

Academic Project

We are Master 2 students at Dakar Institute of Technology (DIT). This project was developed as part of our advanced software engineering coursework, focusing on modern DevOps practices and Machine Learning operations.

Project Objectives

  • Apply source code version control concepts using Git and GitHub for collaborative development
  • Implement Machine Learning data and model versioning using DVC (Data Version Control)
  • Set up a continuous integration (CI) pipeline using Jenkins for automated testing and deployment

Our web application demonstrates these concepts in practice, providing an intuitive interface for training and evaluating machine learning models while maintaining professional standards for version control and continuous integration.
